## Data Requirements

Before uploading data, ensure:
- All spatial data are uniformly projected using WGS 84 or Japan National Coordinate System (JGD)
- Time fields are uniformly formatted (YYYY-MM-DDTHH:mm:ss.sssZ)
- Each layer includes metadata with field definitions, data sources, collection time and unit specifications
- Unified spatial ID fields are used for spatial connection and attribute merging
